# BarberApp

Sistema web de gestión de barbería desarrollado como proyecto académico utilizando tecnologías web básicas y control de versiones con Git y GitHub.

## Descripción

El sistema permite gestionar las operaciones principales de una barbería, facilitando el registro de clientes, administración de citas y control de servicios.

## Tecnologías Utilizadas

* HTML
* CSS
* JavaScript
* Git
* GitHub
* Visual Studio Code

## Funcionalidades Principales

* Registro de clientes
* Gestión de citas
* Administración de servicios
* Visualización de información
* Organización básica del sistema web
